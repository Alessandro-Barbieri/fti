Decides wich action start depending on the ckpt. level.

Parameters
levelCleaning checkpoint level.
groupMust be groupID if App-proc. or 1 if Head.
prMust be 1 if App-proc. or nbApprocs if Head.
Returns
integer FTI_SCES if successful.

This function cleans the checkpoints of a group or a single process. It does that for each group (application process in the node) if executed by the head, or only locally if executed by an application process. The parameters pr determine if the for loops have 1 or number of App. procs. iterations. The group parameter help determine the groupID in both cases.
